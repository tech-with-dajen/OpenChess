# OpenChess
Bold play starts with open code.

Openchess is a modern  and lightweight chess project.
It will contain the following features:
-Chess board ui
-Legal Moves and Board validation
-Figure logic (Rook,Queen and co)
-Check, Checkmate and Stalemate validation
-Clean and modern java-Architecture

It is made for:
-People who enjoy chess and programming
-People who want to learn about chess
-Open source enthusiasts

Installation:
bash:
git clone https://github.com/tech-with-dajen/OpenChess.git
cd OpenChess
